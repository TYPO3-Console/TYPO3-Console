=================
frontend:asseturl
=================


**Show asset URL for TYPO3 extension(s)**

Shows public asset URLs for one or all TYPO3 extensions



Options
=======

`--extension|-e`
   Extension key

- Accept value: yes
- Is value required: yes
- Is multiple: yes
- Default: array ()

`--format|-f`
   Output format (any of "table", "json", "text")

- Accept value: yes
- Is value required: yes
- Is multiple: no
- Default: 'table'

`--composer-name|-c`
   Use composer name as index

- Accept value: no
- Is value required: no
- Is multiple: no
- Default: false
